# MorphoLogic

Learn about common morphological operations in image processing:
- Erosion
- Dilation
- Opening
- Closing

Generate a binary image of a letter or number and apply the morphological operations to it.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
